# Sales Analysis Dashboard (Power BI)

## Project Overview

This project analyzes retail sales performance using Power BI.
It provides insights into sales trends, regional performance, and product analysis.

## Dataset Fields

* Date
* Product
* Category
* Region
* Salesperson
* Customer Segment
* Quantity
* Sales Amount
* Profit
* Target

## Dashboard Features

* KPI Cards (Sales, Profit, Quantity)
* Sales Trend Line Chart
* Region-wise Sales Analysis
* Top Products Identification
* Category Contribution (Donut Chart)
* Salesperson Performance
* Sales vs Profit Scatter Chart
* Interactive Slicers

## Tools Used

* Power BI
* Excel

##  Key Insights

* Identifies top performing regions and products
* Tracks sales trends over time
* Helps in decision-making
